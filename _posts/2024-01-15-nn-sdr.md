## 프로젝트 개요

고차원 예측 변수 $X \in \mathbb{R}^p$와 반응 변수 $Y$ 간의 관계를 신경망으로 학습하여 **충분차원축소(Sufficient Dimension Reduction)**를 수행하는 연구 프로젝트이다. 기존 SIR, SAVE, DR, MAVE 등 역회귀 기반 방법의 선형성 가정 한계를 극복하기 위해, 첫 번째 층 가중치에 **직교 제약**을, 두 번째 층에 **Group Lasso 페널티**를 부과한 신경망 구조를 제안한다.

8개 시뮬레이션 모델에서 고전적 SDR 방법 대비 성능을 검증했으며, 블로그 내 [관련 논문 리뷰 12편](#관련-논문-리뷰)과 연계되는 연구이다.

> 소스 코드는 요청 시 제공 가능합니다.

## 핵심 문제와 해결 접근

### 문제 1: 고전적 SDR 방법의 한계

SIR(Sliced Inverse Regression), SAVE, DR 등 역회귀 기반 방법은 **Linearity Condition**과 **Constant Variance Condition**을 가정한다. 이 가정이 위배되는 비선형 링크 함수나 복잡한 다중 지표 모델에서는 Central Subspace를 정확히 추정하지 못한다. MAVE는 비모수적 접근으로 이 제약을 벗어나지만, 고차원(p > 15)에서 수렴이 느리고 계산 비용이 크다.

| 방법 | 핵심 가정 | 비선형 대응 | 차원 자동 선택 | 고차원 확장성 |
|------|----------|-----------|-------------|------------|
| SIR | Linearity | 제한적 | 불가 | 양호 |
| SAVE | Linearity + Constant Variance | 제한적 | 불가 | 양호 |
| DR | Linearity + Constant Variance | 중간 | 불가 | 양호 |
| MAVE | 없음 (비모수) | 가능 | 불가 | 느림 |
| **NN-SDR** | **없음** | **가능** | **Group Lasso로 가능** | **양호** |

**해결: 신경망이 링크 함수를 직접 학습 + 직교 제약으로 SDR 방향 인코딩**

신경망의 첫 번째 층 가중치 행렬 $W \in \mathbb{R}^{p \times d}$가 Central Mean Subspace의 기저를 인코딩하도록 설계한다. $Y = f(W^T X) + \epsilon$ 관계에서 링크 함수 $f$를 신경망의 나머지 층이 근사하므로, 비선형 관계를 자연스럽게 포착한다. 핵심은 $W$가 직교 기저가 되도록 제약을 부과하는 것이다.

**복합 손실 함수:**

$$L(\theta) = \underbrace{\text{MSE}(y, \hat{y})}_{\text{예측 손실}} + \underbrace{\lambda_0 \| W^T \Sigma W - I_d \|_F^2}_{\text{직교 페널티}} + \underbrace{\lambda_1 \sum_{j=1}^{h} \| w_j^{(2)} \|_2}_{\text{Group Lasso}}$$

- **직교 페널티**: $W^T \Sigma W = I_d$를 강제하여 추정된 방향이 직교 기저를 이루도록 함 ($\Sigma$는 $X$의 공분산 행렬)
- **Group Lasso**: 두 번째 층 가중치를 열 단위로 수축시켜 불필요한 차원을 자동 제거

### 문제 2: 구조적 차원(d)을 모를 때 자동 선택

실제 데이터에서는 축소할 차원 $d$를 사전에 알 수 없다. 기존 방법은 역회귀 행렬의 고유값을 검정(Ladle Estimator 등)하여 $d$를 추정하지만, 신경망 프레임워크에서는 이 접근이 직접 적용되지 않는다.

**해결: Group Lasso로 은닉 유닛 자동 가지치기**

두 번째 은닉층의 가중치 $W^{(2)}$에 Group Lasso 페널티를 부과한다. 학습이 진행되면서 불필요한 열의 가중치가 0으로 수축되어 해당 은닉 유닛이 비활성화된다. 최종적으로 남은 활성 유닛 수가 추정 차원 $\hat{d}$가 된다.

**Proximal Operator (Group Lasso 업데이트):**

$$w_j \leftarrow \left(1 - \frac{\eta \lambda_1}{\|w_j\|_2}\right)_+ w_j$$

여기서 $(\cdot)_+ = \max(\cdot, 0)$이다. $\|w_j\|_2$가 임계값 $\eta \lambda_1$ 이하이면 해당 열이 정확히 0이 되어 차원이 제거된다.

**2단계 최적화:**

| 단계 | 대상 | 방법 | 역할 |
|------|------|------|------|
| Stage 1 | MSE + 직교 페널티 (smooth) | Adam gradient descent | 예측 정확도 + 직교성 확보 |
| Stage 2 | Group Lasso (non-smooth) | Proximal operator | 차원 선택 (가지치기) |

매 에폭마다 Stage 1 → Stage 2를 순차 적용하는 **proximal gradient descent** 방식이다.

### 문제 3: 학습 안정성과 수렴

직교 제약과 Group Lasso를 동시에 최적화하면 학습이 불안정해질 수 있다. 특히 신호 대 잡음비(SNR)가 낮은 모델에서 무작위 초기화로는 수렴이 어렵다.

**해결: DR 초기화 + 적응적 학습률**

```
[1] DR 초기화 (R via rpy2)
    └── Directional Regression 추정치 계산 → W_init (warm start)
        ↓
[2] 학습 루프 (epoch 1..5000)
    ├── Adam step: MSE + λ₀·직교 페널티
    ├── Proximal step: Group Lasso on layer 2
    └── lr 조정: 5 에폭 연속 loss 미개선 시 lr *= 0.9
        ↓
[3] 수렴 판정
    └── loss 변화율 < threshold → 조기 종료
```

- **DR 초기화**: 고전적 DR 방법의 추정치를 첫 번째 층의 초기 가중치로 사용한다. 완전 무작위 대비 수렴 속도와 안정성이 크게 개선된다.
- **직교 제약 변형**: $W^T \Sigma W = I_d$ (공분산 가중)와 $WW^T = I$ (단순 직교) 두 가지를 구현하여, 데이터 특성에 맞는 제약을 선택할 수 있다.

## 네트워크 구조

```
Input(p)
  │
  ├── Linear(p → h₁) ─── 직교 제약: W^T ΣW = I_d
  │         │              (SDR 방향 인코딩)
  │       tanh
  │         │
  ├── Linear(h₁ → h₂) ── Group Lasso 적용
  │         │              (차원 자동 선택)
  │       ReLU
  │         │
  ├── Linear(h₂ → h₂)
  │         │
  │       tanh
  │         │
  └── Linear(h₂ → 1) ─── 최종 예측 ŷ
```

- **1층** ($p \to h_1$): SDR 방향 인코딩. $h_1 = d$(알려진 경우) 또는 $h_1 > d$(미지의 경우, Group Lasso가 가지치기)
- **2층** ($h_1 \to h_2$): Group Lasso 적용 대상. 열 단위 수축으로 유효 차원 결정
- **3~4층**: 링크 함수 $f$ 근사. 충분한 표현력 확보를 위해 2개 층 사용

## 시뮬레이션 설계

### 데이터 생성

$$X \sim N(0, \Sigma), \quad \Sigma_{ij} = 0.5^{|i-j|} \quad \text{(AR(1) 공분산)}$$

- $p = 20$ (예측 변수 차원)
- $n = 500$ (표본 크기)
- $\epsilon \sim N(0, 0.5^2)$ (잡음)
- 모델당 **100회 반복** 실험으로 통계적 강건성 확보

### 8개 시뮬레이션 모델

| 모델 | 차원 | 반응 함수 $Y$ | 특성 |
|------|------|-------------|------|
| M1 | $d=1$ | $(X\beta_1)^2 + \epsilon$ | 이차 함수, SIR 실패 |
| M2 | $d=1$ | $\sin(X\beta_1) + \epsilon$ | 주기 함수 |
| M3 | $d=1$ | $(X\beta_1) / (0.5 + (X\beta_2 + 1)^2) + \epsilon$ | 비선형 비율 |
| M4 | $d=1$ | $(X\beta_1)(X\beta_1 + X\beta_2 + 1) + \epsilon$ | 곱 상호작용 |
| M5 | $d=2$ | $(X\beta_1)^2 + (X\beta_2)^2 + \epsilon$ | 다중 이차 |
| M6 | $d=2$ | $\cos(X\beta_1) \cdot (X\beta_2 + 1) + \epsilon$ | 삼각×선형 |
| M7 | $d=2$ | $(X\beta_1)(X\beta_2) + \epsilon$ | 곱 모델 |
| M8 | $d=2$ | $(X\beta_1)^3 + \sin(X\beta_2) + \epsilon$ | 고차 비선형 |

- **M1~M4**: Single-index 모델 ($d=1$). 링크 함수의 비선형성 수준이 점진적으로 증가
- **M5~M8**: Multi-index 모델 ($d=2$). 두 방향 간의 상호작용 복잡도가 증가
- $\beta_1, \beta_2$는 정규화된 방향 벡터로, 처음 몇 개 성분만 비영

## 평가 지표

| 지표 | 의미 | 최적값 |
|------|------|--------|
| **Multiple Correlation** | 추정 부분공간과 실제 부분공간의 정준상관 합 | 높을수록 좋음 |
| **Frobenius Distance** | 사영 행렬 간 프로베니우스 거리 $\|\|P_{est} - P_{true}\|\|_F$ | 0에 가까울수록 좋음 |
| **Correlation Loss** | 정규화된 상관 손실 $1 - \frac{1}{d}\text{tr}(\cdot)$ | 0에 가까울수록 좋음 |
| **General Loss** | 정규화된 사영 행렬 차이의 제곱 | 0에 가까울수록 좋음 |

100회 반복의 **평균 ± 표준편차**를 리포트하여 추정의 안정성까지 평가한다.

## 실험 환경과 도구

### 하이퍼파라미터 탐색

Ray Tune으로 교차검증 기반 하이퍼파라미터 탐색을 수행한다:

| 하이퍼파라미터 | 탐색 범위 | 역할 |
|-------------|----------|------|
| `h_dim2` | 10, 20, 50 | 은닉층 너비 (표현력) |
| `lambda_0` | 0.01 ~ 1.0 | 직교 페널티 강도 |
| `lambda_1` | 0.001 ~ 0.1 | Group Lasso 강도 (차원 선택 민감도) |
| `lr` | 0.001 ~ 0.01 | 학습률 |

### Streamlit 대시보드 (5페이지)

실험 설정, 학습 모니터링, 결과 비교를 위한 인터랙티브 대시보드를 제공한다. 시뮬레이션 파라미터 조정, 실시간 손실 곡선 모니터링, 고전적 방법과의 비교 차트를 하나의 UI에서 관리한다.

### R 연동 (rpy2)

고전적 SDR 방법(SIR, SAVE, DR, MAVE)의 R 구현체를 Python에서 직접 호출하여 동일 데이터셋에서 공정한 비교를 수행한다. `dr` 패키지의 추정치는 NN-SDR의 초기화에도 활용된다.
